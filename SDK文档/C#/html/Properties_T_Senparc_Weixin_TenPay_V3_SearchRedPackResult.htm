属性
  名称说明
act_name
活动名称（请注意活动名称长度，官方文档提示为32个字符，实际限制不足32个字符）
detail_id
红包单号
err_code
错误代码
err_code_des
错误代码描述
hb_type
红包类型,GROUP:裂变红包,NORMAL:普通红包
hblist
红包领取列表,普通红包只有一项，列表红包可以有很多项
mch_billno
商户订单号（每个订单号必须唯一） 组成：mch_id+yyyymmdd+10位一天内不能重复的数字
mch_id
商户号，微信支付分配的商户号
reason
发送失败原因
refund_amount
红包退款金额
refund_time
红包的退款时间（如果其未领取的退款）
remark
活动描述，低版本微信可见
result_code
业务结果,SUCCESS/FAIL
return_code
返回状态码,SUCCESS/FAIL,此字段是通信标识，非交易标识，交易是否成功需要查看result_code来判断
return_msg
返回信息，如非空，为错误原因,签名失败,参数格式校验错误
send_time
红包发送时间
send_type
发放类型,API:通过API接口发放,UPLOAD:通过上传文件方式发放,ACTIVITY:通过活动方式发放
status
红包状态，SENDING:发放中，SENT:已发放待领取，FAILED：发放失败，RECEIVED:已领取，REFUND:已退款
total_amount
红包总金额（单位分）
total_num
红包个数
wishing
祝福语
